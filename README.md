<h3>Возможности:</h3>
<h4>Дашборд: </h4>
  - Отображение количества оборудования
  - Отображение количества складов
  - Отображение перемещений оборудования
<h4>Вкладка оборудования:</h4>
  - Поиск, учет, добавление оборудования в базу данных, а также установка статуса и назначение инвентарного, серийного номеров.
<h4>Вкладка складов:</h4>
   - Отображение складов и наличие оснащения в них.
		<h5>Вкладка перемещений:</h5>
   - Позволяет отслеживать последние изменения в движении материально-технических средств между складами.
	Включает в себя название оборудования, штрихкод на основе инвентарного номера, перемещение откуда и куда, а также ответственный, который назначил перемещение оборудования.
<h5>Вкладка сканера:</h5>
   - Позволяет быстро найти оснащение при помощи сканирования штрихкода с камеры устройства, либо вручную при вводе инвентарного номера вручную.
